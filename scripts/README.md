

## Development & Usage

This project is a Streamlit dashboard for visualizing and analyzing country-level data using Pandas and Plotly. You can filter data by country, explore different columns, and view interactive plots such as boxplots to better understand trends and distributions.

### Steps to Run

1. Install the required libraries:

   ```bash
   pip install -r requirements.txt
   ```
2. Place your CSV data files in the `data/` folder.
3. Run the app:

   ```bash
   streamlit run app.py
   ```
4. Use the sidebar to select countries or columns to generate plots.
5. Explore and interact with the visualizations directly in your web browser.
